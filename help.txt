#在一切的一切之前，建议大家装好Linux系统，MacOS也是方便的，然而Windows用户体验不佳噢
新手请这样操作：
1.首先点击那个绿色的按钮【Clone or download】，复制出现的那个网址链接
2.其次打开终端（Terminal），输入“git clone https://github.com/cosmologywater/water_group.git” #这个网址就是第一步复制的网址
3.这时候如果你的Linux里面还没有git，可能需要先“sudo apt install git”一下
4.这时候你的电脑目录下就会出现一个名字叫water_group的文件夹（本地仓库）啦！#不信可以ls一下试试
5.然后你就可以进入本地的water_group目录，修改里面的文件，或者添加新的文件（里面的QAs是给大家提问题的！关于编程或者宇宙学的问题都可以！wxm同学已经给大家做了示范啦）
6.修改完毕以后请"vi gitpush.sh"，然后在里面git commit -m后面的引号里，简单描述一下你做了哪些修改 #解释仓库的更新内容是一个好习惯
7.最后“./gitpush.sh”，其间会让你输入用户名和密码(我会发到群里，如果忘记了请询问组内其他的人),如果你是第一次用git，它还会让你输入邮箱和用户名，创建一个自己的git（按照终端给出的提示来操作就好）
如果一切顺利的话，你就可以看到github网站上的仓库已经被你更新啦！

有一切疑问欢迎在组内询问。
对git和github感兴趣的同学可以百度“廖雪峰git教程”，大神手把手教学。

一些git命令：
git pull 用远程仓库更新本地仓库
git add 将文件提交到暂存区
git commit -m "" 将暂存区的文件提交到本地仓库，-m后是对版本改动的注释
git push 用本地仓库更新远程仓库
